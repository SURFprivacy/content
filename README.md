### 2. Usable shortcodes
#### 2.1 Headers
```
# header 1
## header 2
### header 3
#### header 4
##### header 5
###### header 6
```

#### 2.2 Header underline
```
"header 1 underline"
====================

"header 2 underline"
--------------------
```

#### 2.3 Emphasis
```
*italic* or _italic_
**bold** or __bold__
*italic and __bold__*
~~strikethrough~~
```

#### 2.4 Lists
```
* for unordered lists
1, 2, 3 etc. for ordered lists
⋅⋅*, ⋅⋅1 etc. for sub-lists
⋅⋅⋅ for properly indented paragraphcs

E.g.:
1. First ordered list item
2. Second ordered list item
⋅⋅* Unordered sub-list
3. Third ordered list item
⋅⋅⋅Indented paragraph
4. Fourth ordered list item
```

#### 2.5 Links
```
[link text](https://link.tld)
[link text + title](https://link.tld "title")
```

#### 2.6 Images
```
![alt text](https://link.tld/image.ext "title")
```
