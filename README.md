### 2. Usable shortcodes
#### 2.1 Headers
```
# HEADER 1
## HEADER 2
### HEADER 3
#### HEADER 4
##### HEADER 5
###### HEADER 6
```

#### 2.2 Header underline
```
"HEADER 1 UNDERLINE"
====================

"HEADER 2 UNDERLINE"
--------------------
```

#### 2.3 Emphasis
```
*italic* or _italic_
**bold** or __bold__
*italic and __bold__*
~~strikethrough~~
```

#### 2.4 Lists
```
* for unordered lists
1, 2, 3 etc. for ordered lists
⋅⋅*, ⋅⋅1 etc. for sub-lists
⋅⋅⋅ for properly indented paragraphcs

E.g.:
1. First ordered list item
2. Second ordered list item
⋅⋅* Unordered sub-list
3. Third ordered list item
⋅⋅⋅Indented paragraph
4. Fourth ordered list item
```
